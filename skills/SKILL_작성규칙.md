| 항목 | 값 |
|------|-----|
| 유형 | 공통 규칙 |
| 적용 대상 | 모든 라벨 에이전트 |

| 항목 | 기준 |
|------|------|

| 완성도 | 문장이 완결되어야 함 |

|------|------|

## 긴 콘텐츠 처리 (기사/인터뷰)

긴 호흡의 기사나 인터뷰는 다음 구조로 요약합니다:

### 요약 구조

```
[핵심 내용] 1~2문장으로 가장 중요한 메시지

[전개 구조]
- 도입: {무엇을 다루는지}
- 전개: {주요 논점/사례}
- 결론: {핵심 주장/시사점}
```

### 처리 규칙

1. **핵심 내용**: 기사의 가장 중요한 메시지를 1~2문장으로
2. **전개 구조**: 기승전결을 3단계(도입-전개-결론)로 요약
3. **인용문**: 핵심 인용은 "따옴표"로 포함 가능
4. **숫자/데이터**: 구체적 수치는 반드시 포함

## 금지 사항

1. **원문 복붙 금지**: 반드시 재구성
2. **미완성 문장 금지**: `...`로 끝나면 안됨
3. **의미없는 제목 금지**: "읽어보기", "클릭하세요" 등
4. **이모지 사용 금지**: 모든 출력에서 이모지 제외
5. **과도한 수식어 금지**: "매우", "정말", "엄청난" 등 자제
