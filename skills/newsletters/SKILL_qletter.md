## 구조 분석

### 뉴스레터 특징
- 이모티콘이 포함된 제목
- '친절한 마케팅 뉴스' 섹션과 '오다 주웠다' 코너 포함
- 각 아이템은 ◾ 로 시작하는 헤딩으로 구분
- 본문에 마크다운 링크와 CTA 버튼 포함
- 광고 수익 및 AI 인프라 투자 관련 내용이 다수
- 퀴즈 및 구독 안내 등 부가 정보 포함

### 아이템 경계
- 각 아이템은 ◾ 로 시작하는 헤딩 뒤에 빈 줄과 본문으로 구분
- 제목은 헤딩 텍스트, 본문은 이어지는 문단
- 이미지 및 CTA 링크는 본문 중간에 삽입

### 제목 위치
- 제목은 헤딩(◾) 또는 굵은 텍스트 형태로 표시
- 메인 기사 제목은 별도 라인 '챗GPT에 광고 넣는다는데요'
- 미니 기사 제목은 '◾ 오다 주웠다' 뒤에 설명

### 본문 위치
- 제목은 헤딩 바로 뒤에 이어지는 문단
- 본문은 여러 문단으로 구성되며, 마크다운 링크와 인용문이 포함될 수 있음

### 링크 위치
- 본문 중 마크다운 형식 [텍스트](URL) 또는 CTA 버튼 '→ 자세히 보기' 형태로 삽입
- 이미지 링크는 별도 표시

---

## 추출 규칙

1. 헤딩(◾)을 아이템 제목으로 사용
2. 제목은 20~50자, 주어+동사+핵심정보 형태
3. 본문은 300~500자 요약, 핵심사실-배경-영향-시사점 구조
4. 키워드는 명사형 3~5개
5. 링크는 본문에 존재하는 마크다운 링크 중 첫 번째 사용, 없으면 빈 문자열

- 푸터(큐레터 연락처, 구독 해지 링크)
- 광고 배너 및 CTA 버튼 외부 링크
