## 구조 분석

### 뉴스레터 특징
- 섹션별로 Opening Act, Main Act, More... 라벨 사용
- 각 섹션 제목 후 클래식 곡 정보 표기
- 사회적 메시지 강조 (장애인 지원, 접근성)

### 아이템 경계
- 'Opening Act:', 'Main Act:', 'More...:' 텍스트로 시작하는 섹션 구분

### 제목 위치
- 섹션 라벨 다음 줄의 첫 번째 문장 (예: '모든 이들을 위한 음악')

### 본문 위치
- 제목 다음 줄부터 링크 또는 다음 섹션 시작 전까지

### 링크 위치
- '공연 더보기 🎶' 텍스트 뒤의 URL (Main Act에만 존재)

## 추출 규칙

1. 섹션 라벨(Opening Act, Main Act, More...)을 기준으로 아이템 분리
2. 제목은 섹션 라벨 직후 첫 문장으로 추출
3. 본문은 제목 다음 줄부터 링크 또는 섹션 종료 전까지 요약
4. 링크는 '공연 더보기' 라벨 뒤 URL로 추출
