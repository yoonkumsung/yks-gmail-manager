## 구조 분석

### 뉴스레터 특징
- 서울경제진흥원(SBA) 지원사업 모집 공고 알림
- 이모지 카테고리별 지원사업 리스트 구조
- 각 항목: 사업명 + 마감일 + 링크

### 카테고리 (이모지 구분)
- 📗지식재산권
- 🚀창업지원
- 💡오픈이노베이션
- 🦄창조산업
- 🌎유통·마케팅
- 📃기업지원
- 💄뷰티
- 기타 이모지 카테고리 가능

### 아이템 경계
- 각 줄이 하나의 지원사업 (사업명 + 마감일)
- 이모지 카테고리 헤더로 분야 구분
- 마감일은 (~M/D) 또는 (~예산소진시까지) 형태

### 링크 위치
- 각 사업 항목 뒤에 stibee 리다이렉트 URL
- 일부 항목은 링크 없이 제목만

---

## 추출 규칙

1. **모든 카테고리의 모든 지원사업을 개별 아이템으로 추출** — 하나도 빠짐없이
2. 제목: 사업명 + 마감일 포함 (예: "IP나래 프로그램 하반기 2차 지원기업 모집(~6/4)")
3. 요약: 사업명, 분야(카테고리), 마감일, 지원 대상(있으면) 포함. 짧아도 OK (본문에 상세 없으므로).
4. 키워드: 카테고리명, 지원 분야 등
5. 링크 추출

## 특이사항

- 지원사업 알림이므로 사용자에게 마감일이 가장 중요한 정보
- 항목 수가 많으므로(20~40개) 요약은 짧게 유지하되 마감일은 반드시 포함
- 상세 내용이 없으므로 300자 요약 규칙 적용하지 않음 — 있는 내용만 추출
