## 구조 분석

### 뉴스레터 특징
- 하나의 주제(스페이스X IPO)에 대해 심층 분석하는 에세이 형식
- 섹션 제목(##)으로 주요 논점을 구분
- 저자(성륜수)와 발행일이 명시됨
- 이미지 캡션 포함

### 아이템 경계
- 전체 텍스트가 하나의 아이템으로 구성됨. 별도의 아이템 구분 패턴 없음.

### 제목 위치
- '# '으로 시작하는 라인의 텍스트 (예: '# 스페이스X IPO: 투자자가 된다는 것의 의미')

### 본문 위치
- 제목 아래 설명문부터 시작하여 '구독 상세 정보' 섹션 직전까지의 모든 텍스트. 본문 내부는 '##'으로 시작하는 소제목으로 구조화됨.

### 링크 위치
- 이 뉴스레터 본문 내에는 추출할 수 있는 외부 원문 링크(URL)가 없음.

---

## 추출 규칙

1. 전체 내용을 하나의 아이템으로 처리.
2. 제목은 '# '으로 시작하는 라인에서 추출.
3. 본문은 제목 아래 설명문부터 '구독 상세 정보' 직전까지의 모든 텍스트를 요약.
4. 본문 내 이미지 캡션([IMAGE: ...])은 제외하고 요약.

---

## 제외 영역

- '구독 상세 정보' 섹션 이후의 모든 텍스트 (구독자 정보, 저작권, 관리 링크 등)
- 본문 내 이미지 캡션 텍스트 ([IMAGE: ...])
- 표 형식의 헤더 ('| 오리지널 | 브라우저에서 읽기 |')

---

## 특이사항

원문에 외부 기사 링크가 포함되어 있지 않아 link 필드는 빈 문자열로 처리됨. 본문은 심층 분석 기사 형식으로, 핵심 주장과 논거를 구조적으로 요약해야 함.
