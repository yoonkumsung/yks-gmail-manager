## 구조 분석

### 뉴스레터 특징
- 섹션별 대문자 제목 구분 (FUNDING, GLOBAL NEWS, MORE NEWS)
- '=' 접두사로 글로벌 뉴스 항목 식별
- 표를 활용한 정부 지원 사업 상세 설명

### 아이템 경계
- 대문자 섹션명(FUNDING, GLOBAL NEWS) 시작 블록
- '=' 기호로 시작하는 문장 단위 (글로벌 뉴스)
- MORE NEWS 내 개별 문단 단위

### 제목 위치
- 섹션 제목 바로 다음 줄 첫 문장 (예: '탄소중립 사업화 사업 참여 기업 모집')
- '=' 뒤 첫 문장 (예: '=오픈AI, 챗GPT 광고 게재 테스트')

### 본문 위치
- 제목 다음 줄부터 다음 섹션 시작 전까지
- 표가 포함된 경우 표 내용과 설명 텍스트 모두 포함

### 링크 위치
- '이곳 [URL]' 패턴의 텍스트 뒤 URL
- 섹션 내 '리포트는 이곳 [URL]' 형식

## 추출 규칙

1. 정부 지원 사업은 구분별 지원 규모 요약

3. 글로벌 뉴스는 핵심 내용 2문장으로 압축
