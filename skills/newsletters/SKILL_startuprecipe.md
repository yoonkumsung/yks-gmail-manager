## 구조 분석

### 뉴스레터 특징
- 매일 발행되는 스타트업/벤처 뉴스 다이제스트
- 4개 주요 섹션으로 구성: 헤드라인 → FUNDING → GLOBAL NEWS → MORE NEWS
- 각 섹션에 다수의 뉴스 아이템 포함

### 섹션 구조 (순서대로)

1. **헤드라인 기사** (1~3개)
   - [DailyRecipe] 접두사 + 제목 + 날짜
   - 상세 본문 포함 (배경, 수치, 분석)
   - 관련 기사 링크 포함

2. **FUNDING** 섹션
   - "FUNDING" 헤더로 시작
   - 각 투자건이 개별 항목: 기업명 → 사업 분야 → 투자금액 → 상세설명
   - 보통 5~10개 투자 뉴스
   - 팁스(TIPS) 선정, 시드/시리즈 투자 등

3. **GLOBAL NEWS** 섹션
   - "GLOBAL NEWS" 헤더로 시작
   - 해외 스타트업/벤처 뉴스 제목 + 링크
   - 보통 3~5개

4. **MORE NEWS** 섹션
   - "MORE NEWS" 헤더로 시작
   - 국내 스타트업 소식 제목 + 링크
   - 보통 10~15개

### 아이템 경계
- 헤드라인: 빈 줄로 기사 구분
- FUNDING: 기업명(줄) → 사업분야(줄) → 금액(줄) → 설명문단
- GLOBAL/MORE NEWS: 각 줄이 하나의 뉴스 (제목 + URL)

### 링크 위치
- 대괄호 안 URL: [https://...]
- "관련 기사 :" 라벨 뒤에도 URL

## 추출 규칙

1. **모든 섹션의 모든 아이템을 빠짐없이 추출** — FUNDING, GLOBAL NEWS, MORE NEWS 포함
2. 헤드라인 기사: 300~500자 상세 요약 (핵심사실+수치+배경+시사점)
3. FUNDING 각 건: 기업명, 사업분야, 투자금액, 투자자를 반드시 포함
4. GLOBAL NEWS: 제목과 핵심 내용 (있으면)
5. MORE NEWS: 제목과 핵심 내용 (제목만 있으면 짧은 요약 허용)
6. 제목 20~50자, 키워드 3~5개
7. 원문 URL 추출

- 날짜 (YYYY.MM.DD)
- 하단 구독/수신거부
- 광고 배너

## 특이사항

- 아이템 수가 많음 (20개+) — 하나도 누락하지 말 것
- FUNDING 섹션의 기업명-분야-금액 구조를 정확히 파악하여 추출
- GLOBAL NEWS와 MORE NEWS는 제목+링크만 있는 경우가 많으므로, 제목 기반 짧은 요약 허용
