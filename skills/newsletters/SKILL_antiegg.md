## 구조 분석

### 뉴스레터 특징
- 다양한 문화/예술 주제 다변화
- 서사적 제목 구성
- 독자 참여 유도형 서문

### 아이템 경계
- '읽으러 가기' 또는 '확인하기'로 끝나는 블록
- 2줄 제목 + 본문 구조

### 제목 위치
- 볼드체 없이 줄바꿈으로 강조된 2줄 제목

### 본문 위치
- 제목 아래 2~4문장 설명문

### 링크 위치
- 본문 종료 직후 '읽으러 가기/확인하기' 텍스트 뒤 URL

---

## 추출 규칙

1. 2줄 제목을 하나의 제목으로 통합
2. 본문 첫 문장에서 핵심 콘셉트 추출
3. '읽으러 가기' 직전까지를 요약 범위

- 푸터 광고/구독 링크
- INSIGHT 섹션 하위 항목
