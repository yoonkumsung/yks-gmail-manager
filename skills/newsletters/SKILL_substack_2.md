## 구조 분석

### 뉴스레터 특징
- 작성자별 독립된 아이템 구성
- 각 아이템에 게시일자 표시
- 반응 수(좋아요/댓글) 표시

### 아이템 경계
- 작성자 이름으로 구분
- 'Read More'로 종료

### 제목 위치
- 작성자 이름 바로 아래 첫 문장

### 본문 위치
- 작성자 이름 아래 1~3문장 요약

### 링크 위치
- 'Read More' 텍스트 링크

---

## 추출 규칙

1. 작성자 이름 기준 아이템 분리
2. 첫 문장을 제목으로 추출
3. 본문은 2~3문장 요약
4. 'Read More' 링크 추출
