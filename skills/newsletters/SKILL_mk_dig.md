## 구조 분석

### 뉴스레터 특징
- 구분선(┃)으로 아이템 분리
- 각 아이템 시작에 제목 강조
- 3줄 요약 제공

### 아이템 경계
- '┃' 기호로 아이템 구분
- 제목 다음 본문 내용 배치

### 제목 위치
- 아이템 시작 부분에 강조 표시
- 제목 바로 아래 본문 시작

### 본문 위치
- 제목 아래 연속된 문단
- 링크는 본문 중간 또는 끝에 삽입

### 링크 위치
- 본문 중간 URL 삽입
- 하이퍼링크 형태로 제공

1. '┃' 기호 기준으로 아이템 분할
