## 구조 분석

### 뉴스레터 특징
- 정부 부처별 정책 자료와 연구 보고서 제목 나열
- 섹션별 분류(경제일반, 금융·통화 등)로 구성
- 기관명을 괄호 안에 표기하는 통일된 형식

### 아이템 경계
- 섹션 제목('경제일반', '금융·통화' 등) 다음 줄부터 '더보기 >' 라인 직전까지
- 각 아이템은 개별 줄 단위로 구분, 기관명 패턴(예: '(재정경제부)')으로 종료

### 제목 위치
- 괄호('(') 앞의 텍스트 전부, 예: '2026년 2월 최근 경제동향'
- 영문 제목은 기관명 앞 콜론(:) 이전 텍스트

### 본문 위치
- 제목 라인 자체가 유일한 콘텐츠로, 별도 본문 없음
- 기관명 표기 직후 빈 줄에서 종료

### 링크 위치
- 개별 아이템에 링크 없음
- 섹션별 '더보기 >' 라인에 전체 링크 집합

---

## 추출 규칙

1. 기관명 제외한 텍스트를 제목으로 추출
2. 영문 제목은 한국어로 완전 번역
3. 중복 아이템은 최초 등장 섹션 기준 추출
