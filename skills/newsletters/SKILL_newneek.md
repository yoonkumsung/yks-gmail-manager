## 구조 분석

### 뉴스레터 특징
- 2030 세대 타겟 시사/문화 뉴스레터
- 상단에 해시태그 키워드 요약
- "오늘의 비트 모아보기" 목차 → 각 섹션 상세 기사
- 섹션별 고유 이름: "비욘드 트렌드", "주말토리", "에디터스 노트"

### 섹션 구조
1. **해시태그 키워드** (상단 1줄) — 오늘의 주제 요약
2. **오늘의 비트 모아보기** — 목차 (추출 대상 아님, 구조 파악용)
3. **비욘드 트렌드 (1)** — 메인 트렌드 기사
4. **비욘드 트렌드 (2)** — 보조 트렌드 기사
5. **주말토리** — 주말 문화생활 추천
6. **에디터스 노트** — 에디터 에세이
7. **#광고** — 광고 (제외)

### 아이템 경계
- "비욘드 트렌드", "주말토리", "에디터스 노트" 키워드가 섹션 시작
- 각 섹션 제목 뒤에 이모지와 부제목

### 링크 위치
- 기사 내 하이퍼링크 텍스트

## 추출 규칙

1. **비욘드 트렌드 (1), (2), 주말토리, 에디터스 노트 모두 빠짐없이 추출**
2. 각 기사 300~500자 요약
3. 광고(#광고) 섹션 제외
4. 제목 20~50자, 키워드 3~5개
5. "오늘의 비트 모아보기"는 목차일 뿐 → 추출하지 않고 하단 상세 기사를 추출
