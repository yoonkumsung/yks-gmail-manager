## 구조 분석

### 뉴스레터 특징
- 각 아이템은 카테고리와 제목으로 시작
- 이모지 사용으로 시각적 강조
- '자세히 보기' 링크 포함

### 아이템 경계
- 카테고리(세계, 테크 등)와 제목으로 구분
- 구분선 없이 텍스트로만 구분

### 제목 위치
- 카테고리 다음 줄에 제목 표시
- 강조 표시 없이 텍스트만으로 표현

### 본문 위치
- 제목 다음 문단에 본문 내용
- 간결한 설명 후 '자세히 보기' 링크

### 링크 위치
- 본문 끝에 '자세히 보기' 텍스트 링크
- 일부 아이템은 본문 내 추가 링크 포함
