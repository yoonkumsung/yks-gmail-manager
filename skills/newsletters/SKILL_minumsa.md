## 구조 분석

### 뉴스레터 특징
- 에디터의 개인적 근황과 감상을 담은 도입부로 시작
- 책 소개 시 긴 인용문(블록 인용)을 포함하여 깊이 있는 내용 전달
- 각 섹션은 이모지(📕, 📗, 🗯️)와 은 텍스트로 구분
- 뉴스레터 말미에 구독 유도 및 피드백 요청 링크 배치
- '한편 담벼락'이라는 커뮤니티 링크가 고정적으로 포함됨

### 아이템 경계
- 책 소개 섹션: '📕[『책제목』](링크)' 패턴으로 시작
- 팟캐스트/미디어 추천: '📗' 이모지와 함께 굵은 텍스트로 시작
- 짧은 서평/추천: '🗯️' 이모지와 함께 인용문 형태로 시작
- 에디터 코멘트: '아아……'와 같은 감탄사로 시작하는 문단

### 제목 위치
- 책 제목: '📕[『제목』](링크)' 또는 '📕『제목』' 패턴 내의 텍스트
- 팟캐스트/미디어: '📗' 이모지 뒤의 굵은 텍스트
- 서평 대상: '🗯️' 이모지 뒤의 첫 문장 또는 링크 텍스트

### 본문 위치
- 책 소개: 제목 링크 다음 문장부터 다음 이모지 섹션(📗, 🗯️) 또는 '― 저자, 『책제목』' 인용 출처 표기 전까지
- 인용문: 들여쓰기된 블록 형태로, '― 저자' 표기로 종료
- 에디터 코멘트: '아아……'로 시작하여 다음 섹션 경계까지

### 링크 위치
- 책 링크: '📕[『책제목』](URL)' 형태로 제목에 직접 포함
- 팟캐스트/미디어 링크: '📗' 섹션 내 '들으러 오세요' 앞뒤의 URL
- 서평 링크: '🗯️' 섹션 내 인용문 앞뒤의 URL
- 구독/피드백 링크: 말미의 '💝', '📕', '💌' 이모지 뒤의 링크

## 추출 규칙

1. '📕[『책제목』](링크)' 패턴을 찾아 책 제목, 본문, 링크를 추출한다.
2. 본문은 제목 다음 문장부터 '― 저자' 인용 출처 전까지의 텍스트를 요약한다.
3. '📗' 섹션에서 팟캐스트/미디어 제목과 링크를 추출한다.
4. '🗯️' 섹션에서 서평 대상과 링크를 추출한다.
5. 에디터 코멘트에서 소개된 책 정보를 추출한다.

- '제철 산딸기'와 같은 단순 감상문
