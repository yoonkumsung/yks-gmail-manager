## 구조 분석

### 뉴스레터 특징
- 두 개의 주요 아이템으로 구성
- 각 아이템 시작 전 디자인 크레딧 표시

### 아이템 경계
- 'design by 슝슝 (w/ChatGPT)' 구분선 사용
- 숫자 없는 제목 헤드라인으로 아이템 시작

### 제목 위치
- 아이템 시작 부분에 볼드체 없는 텍스트 제목
- '🥤' 이모티콘으로 시각적 강조

### 본문 위치
- 제목 직후 연속된 문단으로 본문 구성
- 본문 중간에 관련 링크 포함

### 링크 위치
- 본문 내 인라인 하이퍼링크 형태
- 참고 자료 링크로 사용

## 추출 규칙

1. 'design by' 구분선 이후 내용을 새 아이템으로 구분
2. 제목은 첫 문장에서 추출
3. 본문은 3~5개 핵심 문단으로 요약
