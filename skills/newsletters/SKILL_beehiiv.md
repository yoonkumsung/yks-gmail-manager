## 구조 분석

### 뉴스레터 특징
- 아이템별로 이모지(🔥, 🎨, 🤖 등)로 시작하는 제목 라인
- 소속, 핵심 키워드, 질문, 설명, 주목할 점, 게임 체인저 섹션 고정 구조
- 3줄 요약 섹션으로 본문 시작

### 아이템 경계
- 이모지(🔥, 🎨, 🤖, ⚡, 🔬 등)로 시작하는 라인

### 제목 위치
- 이모지 뒤 따옴표 없는 제목 텍스트 (예: "이제 모델 크기는 중요하지 않다! 자연어 피드백 한 줄의 위력")

### 본문 위치
- 제목 다음 줄부터 '🎯 왜 이것이 게임 체인저인가?' 섹션 직전까지의 소속/핵심 키워드/질문/설명/주목할 점

### 링크 위치
- '🔗 더 자세한 내용이 궁금하다면 :' 라벨 뒤 URL

---

## 추출 규칙

1. 이모지 접두사 라인에서 제목 추출
2. '🏛️ 소속'부터 '🎯' 직전까지 본문 요약 생성
3. '🔗' 라벨 뒤 URL만 링크로 추출

---

## 제외 영역

- 헤더(날짜, 회사 목록)
- '📈 최신 AI 트렌드 3줄 요약' 섹션
- 푸터(저작권, 이용약관, 이미지 설명)
