## 구조 분석

### 뉴스레터 특징
- 첫 번째 아이템은 메인 기사 (본문이 길고 상세)
- 이후 보조 기사들이 이어짐
- 모든 아이템은 "> 자세히 보기" 링크로 종료

### 아이템 경계
- 아이템 종료: "> 자세히 보기" 텍스트
- 첫 번째 아이템: 날짜(예: "2026. 5. 21") 이후 본문부터 첫 "> 자세히 보기"까지
- 이후 아이템: 볼드 제목 행으로 시작, "> 자세히 보기"로 종료

### 제목 위치
- 첫 번째 아이템: 메일 제목 또는 본문 첫 볼드 텍스트
- 이후 아이템: 볼드 처리된 제목 행 (예: "국고채 1조원어치 사들인 '큰손'은 누구?")

### 본문 위치
- 제목 이후부터 "> 자세히 보기" 직전까지

### 링크 위치
- "> 자세히 보기" 텍스트에 포함된 URL

## 추출 규칙

1. "> 자세히 보기"를 아이템 종료 경계로 사용
2. 첫 번째 블록 = 메인 기사 (날짜 이후부터 첫 "> 자세히 보기"까지)
3. 이후 블록 = 각각 볼드 제목부터 다음 "> 자세히 보기"까지
4. 각 아이템의 "> 자세히 보기" URL 추출
