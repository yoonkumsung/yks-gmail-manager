## 구조 분석

### 뉴스레터 특징
- 여러 주제를 다루는 멀티 아이템 뉴스레터
- 각 섹션에 굵은 제목(strong/h2) 사용
- 기사 인용과 CTA 포함
- 소셜 미디어 및 구독 안내 등 부가 정보 포함

### 아이템 경계
- 섹션 제목은 굵은 텍스트(strong) 또는 h2 스타일로 구분
- 제목 뒤에 빈 줄과 본문(p) 텍스트가 이어짐
- 섹션 간 빈 줄과 구분선으로 구분

### 제목 위치
- 각 아이템의 제목은 섹션 첫 번째 라인, strong 태그 또는 h2 스타일로 표시

### 본문 위치
- 제목은 빈 줄 뒤에 이어지는 p 태그 형태의 본문 텍스트

### 링크 위치
- 기사 인용은 '기사: "제목": URL' 형태로 본문에 포함
- 하단 소셜 미디어 링크는 별도 섹션에 배치

---

## 추출 규칙

1. 섹션 제목을 아이템 제목으로 추출
2. 제목 뒤 본문을 300~500자 요약
3. 기사 인용은 원문 URL이 없으므로 빈 문자열 처리
4. 헤더·푸터·광고 영역은 제외
5. 키워드는 명사형으로 3~5개 선정
