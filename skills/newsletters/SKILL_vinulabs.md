## 구조 분석

### 뉴스레터 특징
- 이모지(🍗, 🍔, 🛵)로 시작하는 섹션 제목 라인
- 섹션 제목 다음 줄에 굵은 부제목(**볼드**) 라인
- 각 섹션은 인용문(💬)과 분석 본문으로 구성
- 말미에 'Full Report 보러가기' CTA 링크 존재
- 상단에 구독/지난레터/문의 링크, 하단에 지난레터/공유 링크 영역

### 아이템 경계
- 이모지(🍗, 🍔, 🛵 등)로 시작하는 라인이 새로운 아이템의 시작
- 이모지 라인 이전의 빈 줄과 이후의 굵은 부제목 라인이 경계를 형성

### 제목 위치
- 이모지 라인 바로 다음 줄에 위치한 **볼드** 텍스트 (예: 'BHC, 다른 치킨 기죽이는 압도적 선호도')
- 이모지 라인 자체도 제목 성격을 가지나, 볼드 라인이 실질적인 기사 제목 역할

### 본문 위치
- 볼드 부제목 다음 줄부터 다음 이모지 섹션 시작 전까지의 모든 텍스트
- 인용문(💬)과 분석 단락 포함
- 'Full Report 보러가기' 링크 직전까지

### 링크 위치
- 각 섹션 말미에 별도 링크 없음
- 전체 뉴스레터 말미의 'Full Report 보러가기' 링크가 전체 리포트 URL
- 상단의 구독/지난레터/문의 링크는 제외

## 추출 규칙

1. 이모지(🍗, 🍔, 🛵 등)로 시작하는 라인을 찾아 섹션을 분리한다
2. 각 섹션의 볼드 부제목을 제목으로 사용한다
3. 인용문(💬)과 분석 단락을 합쳐 요약을 생성한다
4. 말미의 'Full Report 보러가기' 링크를 모든 아이템의 공통 링크로 사용한다
5. 상단 메뉴, 하단 지난레터/공유 영역은 제외한다

---

## 제외 영역

- 상단의 '구독하기', '지난 레터 보러가기', '서비스 문의하기' 링크 영역
- 첫 인사말 및 도입부 (⚽️ 이모지로 시작하는 단락부터 🍗 섹션 전까지)
- 말미의 '다음 편은?' 예고 영역
- '금성윤님, 지난 Z세대 트렌드 레터가 궁금하신가요?' 및 '보러가기' 링크
- 'Z세대 트렌드 레터 구독링크 공유하기' 및 VINU(주) 정보

## 특이사항

- 각 섹션이 이모지+볼드부제목+인용문+분석의 일관된 구조
- 인용문(💬)은 실제 Z세대 인터뷰를 인용한 형식
- 전체 리포트 링크가 하나만 존재하며, 개별 기사 링크는 없음
- '다음 편은?' 코너에서 다음 호 예고
