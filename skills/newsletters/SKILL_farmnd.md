## 구조 분석

### 뉴스레터 특징
- 지역 기반(곡성) 청년들의 삶을 다루는 매거진 형식의 뉴스레터
- 웹 매거진 본문으로 유도하는 티저 형태의 구성
- 'TOP 5'와 같은 리스트 형식을 활용하여 가독성 확보
- 독자 참여형 이벤트 섹션을 포함하여 양방향 소통 지향

### 아이템 경계
- 숫자로 시작하는 리스트 패턴 (예: 1., 2., 3.)
- 숫자 뒤에 이모지와 함께 배치된 짧은 문장

### 제목 위치
- 숫자와 이모지 바로 뒤에 위치한 핵심 질문 문구

### 본문 위치
- 제목 다음 줄부터 '(더보기)' 또는 다음 번호가 나오기 전까지의 텍스트

### 링크 위치
- '(더보기)' 텍스트 뒤의 대괄호([]) 안에 포함된 URL
- '전체 내용 보러가기'와 같은 CTA 버튼 옆의 URL

## 추출 규칙

1. 번호가 매겨진 각 질문을 개별 아이템으로 추출
2. 이벤트 섹션은 별도의 참여 정보 아이템으로 추출
3. 본문 텍스트가 짧은 경우 뉴스레터 도입부의 맥락을 활용하여 요약문 작성
