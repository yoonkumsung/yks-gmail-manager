## 구조 분석

### 뉴스레터 특징
- 편집자의 개인적인 소개와 인사말 포함
- 뉴스 아이템은 제목과 본문으로 구성

### 아이템 경계
- 'News || Views' 섹션 이후 아이템 나열
- 각 아이템은 대괄호 섹션 마커로 시작 (예: [커머스BN], [AI], [테크BN], [미디어BN] 등)
- 대괄호 섹션 마커가 나타날 때마다 새로운 아이템의 시작
- 각 아이템은 제목과 본문으로 구분
- 하나의 섹션 마커 아래 여러 기사가 있을 수 있으며, 각 기사는 개별 아이템으로 추출

### 제목 위치
- 대괄호 섹션 마커 뒤 또는 아래에 위치 (예: [커머스BN] 제목텍스트)
- 하이퍼링크 텍스트로 제목 표시

### 본문 위치
- 제목 아래 본문 텍스트

### 링크 위치
- 제목 하이퍼링크

## 추출 규칙

1. 'News || Views' 섹션 이후 아이템 추출
2. 대괄호 섹션 마커([커머스BN], [AI] 등)를 아이템 경계로 인식
3. 각 섹션 마커 아래의 모든 기사를 개별 아이템으로 추출 (약 10개)
4. 제목은 하이퍼링크 텍스트
5. 본문은 제목 아래 텍스트
