## 구조 분석

### 뉴스레터 특징
- 섹션 헤더에 이모지와 한영 병기 라벨 사용
- 요약 목록 후 상세 설명 구조
- 각 아이템별 '지원하기'/'더보기' CTA 배치

### 아이템 경계
- '섹션 제목 | 영문 라벨' 형식의 헤더 다음 이모지로 시작하는 제목 라인
- 예: '📆 재단 소식 & 일정 | ANF UPDATE' 후 '🚀 아산 보이저 2026 배치 모집'

### 제목 위치
- 섹션 헤더 직후 이모지(🚀, 📝 등)로 시작하는 라인의 텍스트
- 예: '🚀 아산 보이저 2026 배치 모집 (~3/22)'

### 본문 위치
- 제목 다음 줄부터 다음 아이템 제목 또는 섹션 헤더 직전까지의 문단
- '지원하기'/'더보기' 문구 이전까지

### 링크 위치
- '지원하기', '더보기' 라벨 직전에 배치된 URL
- 예: '지금 바로 아산 보이저에 지원하세요! 🙌🏻 지원하기'

## 추출 규칙

1. 섹션 헤더 기준 아이템 그룹화
2. 이모지로 시작하는 라인을 제목으로 추출
3. 문단 끝까지 본문으로 처리
4. CTA 텍스트 직전까지 요약 작성
