## 구조 분석

### 뉴스레터 특징
- 이모지(🌊, 🎬, 🏢)로 시작하는 아이템 구분
- 섹션 헤더 대문자 표기(ANF UPDATE, INSIGHT REPLAY)
- '더보기' 링크를 통한 상세 콘텐츠 연결

### 아이템 경계
- 이모지(🌊, 🎬, 🏢)로 시작하는 라인
- '현장속으로' 표기된 워크숍 섹션 구분

### 제목 위치
- 이모지 직후 텍스트 (예: '🌊 단기 아산 프론티어 아카데미 수강생 모집')
- 섹션 내 첫 번째 문장

### 본문 위치
- 제목 다음 줄부터 '더보기' 또는 CTA 텍스트 직전까지

### 링크 위치
- '더보기' 라벨 뒤 URL
- '📝 지원하기', '👩🏻‍💻 신청하기' CTA 텍스트 뒤 링크

---

## 추출 규칙

1. 이모지 패턴으로 아이템 경계 식별
2. 제목은 이모지 직후 50자 이내 텍스트로 추출
3. 본문은 핵심 내용 300~500자로 재구성
