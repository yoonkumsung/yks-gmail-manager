## 추출 규칙

**핵심: 이 뉴스레터는 정부 정책 보도자료 제목 목록이다. 각 제목 = 1개 아이템.**

1. 모든 정책/보고서 제목을 각각 1개 아이템으로 추출
2. 중복 제거: 같은 제목이 여러 카테고리에 반복되면 1개만 추출
3. 섹션 헤더("경제정책자료", "국내연구자료", "해외경제동향" 등)는 아이템이 아닌 카테고리
4. link 필드: eiec.kdi.re.kr URL 반드시 추출 (kdi.re.kr/accesslog 리다이렉트 URL은 실제 URL로 변환)
5. 키워드 2~3개

### 요약 작성법 (중요)
- 제목만 있으므로 **제목을 완성된 한국어 문장으로 재구성**
- 부처명(괄호 안)을 주어로, 정책 내용을 서술어로 풀어쓰기
- 예: "물그릇은 크게, 홍수 예측은 정확하게 (기후에너지환경부)"
  → "기후에너지환경부가 저수 시설 확대와 홍수 예측 정확도 향상 종합대책을 발표했다."
- 제목에 없는 내용 절대 추가 금지. 제목의 의미를 풀어쓰기만 할 것.
