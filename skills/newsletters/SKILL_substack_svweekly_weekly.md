## 구조 분석

### 뉴스레터 특징
- 단일 주제 심층 분석
- 서브스택 뉴스레터 고유 서식
- 섹션별 소제목 계층 구조

### 아이템 경계
- 본문 전체가 단일 아이템으로 구성되어 별도 경계 없음

### 제목 위치
- 메일 본문 상단 첫 번째 제목 라인 (예: '결국 음성을 지배하는 자가 AI를 지배합니다: OpenAI의 새로운 '귀'')

### 본문 위치
- 제목 직후부터 'Share' 텍스트 시작 전까지의 연속된 단락

### 링크 위치
- 'Read full story' 텍스트 뒤에 위치한 URL (clean_text 내 실제 URL 미표기)

## 추출 규칙

1. 단일 주제 심층 분석 뉴스레터로 본문 전체를 하나의 아이템으로 처리
2. 제목은 20~50자 내외의 핵심 메시지로 재구성
3. 요약은 핵심 주장-근거-전망 구조로 300~500자 작성
