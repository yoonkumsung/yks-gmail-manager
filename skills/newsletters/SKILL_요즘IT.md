# SKILL: 요즘IT

위시켓이 운영하는 개발자/IT 종사자 대상 뉴스레터.

---

## 메타 정보

```yaml
from_patterns:
  - "yozm_help@wishket.com"
  - "wishket.com"
  - "요즘IT"
type: single_feature
language: ko
frequency: daily
```

---

## 뉴스레터 구조

```
┌─ 헤더 ─────────────────────────────┐
│ 날짜 | '출근길' 루틴               │
└─────────────────────────────────────┘
           │
┌─ 피처 기사 ────────────────────────┐
│ 메인 제목                           │
│ 기사 요약/소개                      │
│ "자세히 보기" 링크                  │
└─────────────────────────────────────┘
           │
┌─ 부가 콘텐츠 (선택) ───────────────┐
│ 추가 기사/팁                        │
└─────────────────────────────────────┘
           │
┌─ 푸터 ─────────────────────────────┐
│ 위시켓 소개                         │
│ 수신거부                            │
└─────────────────────────────────────┘
```

---

## 특징

- **하나의 피처 기사** 중심
- 개발자 커리어, 기술 트렌드, 인터뷰 등
- 실용적인 정보 제공

## 추출 규칙

|------|------|

### 제외
| 유형 | 이유 |
|------|------|
| 위시켓 홍보 | 서비스 홍보 |
| 구인/구직 안내 | 광고성 |

---

## 제목 처리

요즘IT 제목은 종종 길거나 질문 형식:

```
원본: "프랑스 개발자는 왜 한국에서 테크 채용 플랫폼을 만들게 되었을까?"

처리:
- 질문 형식 유지 가능
- 또는 핵심 내용으로 변환: "프랑스 개발자의 한국 테크 채용 플랫폼 창업기"
```

---

```
프랑스 출신 개발자 줄리앙이 한국에서 테크 채용 플랫폼을 창업한 이야기를 들어봤습니다. 그는 한국의 개발자 채용 시장에서 발견한 문제점과 이를 해결하기 위한 접근 방식을 공유합니다.

자세히 보기
```
