## 구조 분석

### 뉴스레터 특징
- 메인 기사 1편 + 하단 가상자산 시황 데이터로 구성
- AI가 작성한 뉴스레터 (Claude, ChatGPT 활용)
- 인사말("안녕하세요 엠블록레터의 AI 인턴🤖입니다")로 시작

### 아이템 경계
- 아이템 1 (메인 기사): 인사말부터 "📊 가상자산 시황" 직전까지
- 아이템 2 (시황): "📊 가상자산 시황"부터 "Written by" 직전까지

### 제목 위치
- 메인 기사: 메일 제목 또는 인사말 직후의 볼드/강조 텍스트
- 시황: "📊 가상자산 시황"

### 본문 위치
- 메인 기사: 인사말 이후 본문 전체 (매우 길 수 있음)
- 시황: 시세 데이터 및 시장 요약

### 링크 위치
- 본문 내 인라인 링크

---

## 추출 규칙

1. 메인 기사 = 인사말("안녕하세요")부터 "📊 가상자산 시황" 직전까지 전체를 1개 아이템으로 추출
2. 시황 섹션 = "📊 가상자산 시황"부터 "Written by" 직전까지를 1개 아이템으로 추출
3. 메인 기사의 제목은 메일 제목에서 추출
