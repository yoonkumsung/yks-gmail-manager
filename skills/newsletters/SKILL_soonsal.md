## 구조 분석

### 뉴스레터 특징
- 이모지로 시작하는 섹션 제목
- 각 아이템 별 제목-본문-링크 구조
- 섹션 간 구분선(---)으로 구분
- 각 섹션 시작에 이모지 아이콘 사용
- 본문 내 숫자와 데이터 중심 구성

### 아이템 경계
- 이모지(🛡️, ⚖️, 🎰 등)로 시작하는 라인으로 섹션 구분

### 제목 위치
- 이모지 뒤 대문자 섹션명(60/40의 신화, 연준을 둘러싼 권력 싸움-ing 등)

### 본문 위치
- 제목 다음 줄부터 '◾' 마크로 시작하는 소제목 구조

### 링크 위치
- 각 섹션 끝에 '원문' 라벨 뒤 URL

---

## 추출 규칙

1. 이모지로 시작하는 섹션을 하나의 아이템으로 인식
2. 섹션명을 제목으로 추출
3. 본문은 '◾' 마크로 시작하는 소제목 구조 포함
4. 원문 URL을 링크로 추출
