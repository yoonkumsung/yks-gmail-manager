## 아이템 경계 식별

### 주제 시작 신호
- 이모지 + 제목: `🦾아마존 '프로젝트 던'이...`
- 굵은 제목: `**주제 제목**`
- TOC에 나열된 항목

### 주제 종료 신호
- 다음 이모지 제목
- `👉` CTA 링크
- `[브랜디드]` 광고 시작
- 빈 줄 + 새로운 섹션

### 포함 (뉴스 가치 있음)

## 링크 처리

### 본문 내 링크
```
원본: 👉 AI 시대 화이트칼라의 위기... 한국은?
처리:
{
  "text": "AI 시대 화이트칼라의 위기... 한국은?",
  "href": "https://themiilk.com/articles/..."
}
```

### 추적 파라미터 제거
```
제거: utm_source, utm_campaign, utm_medium, utm_term
예: ?utm_source=Viewsletter&utm_campaign=xxx → 제거
```

## 아이템 통합 규칙

### 같은 아이템으로 통합
```
"1만 6000명" + "아마존이 발표한 감원 규모입니다" + "AI 전환에 따른..."
→ 모두 같은 아마존 감원 뉴스 → 하나의 아이템
```

### 별도 아이템으로 분리
```
"아마존 감원" vs "AI 산업 효율화" vs "화이트칼라 긱 이코노미"
→ 각각 다른 주제 → 별도 아이템
```
