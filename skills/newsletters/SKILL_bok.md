## 구조 분석

- 'BOK 경제연구' 섹션에서는 [[제XXXX-XX호] 제목] 패턴으로 새 논문 아이템이 시작되며, 저자 정보가 괄호 안에 표시됨
- '최신 해외학술 정보' 섹션에서는 "[“논문명”]" 형식의 따옴표 안에 제목이 위치하고, 다음 줄에 발행기관 및 저자 정보가 나열
- 각 아이템 뒤에는 원문 링크가 URL 형태로 명시됨

### 제목 위치
- 'BOK 경제연구' 섹션: [[제XXXX-XX호] 뒤에 오는 영문 제목 텍스트
- '최신 해외학술 정보' 섹션: [“와 ”] 사이의 텍스트

### 본문 위치
- 'BOK 경제연구' 섹션: 제목과 저자 정보 다음 줄부터 다음 [[제XXXX-XX호] 패턴 또는 섹션 종료 전까지의 전체 영문 요약 텍스트
- '최신 해외학술 정보' 섹션: 논문 제목과 발행기관/저자 정보만 제공되며, 별도의 본문 요약은 없음 (제목만 나열된 경우에는 본문 없음으로 간주)

### 링크 위치
- 'BOK 경제연구' 섹션: 제목 텍스트 앞에 하이퍼링크로 제공된 URL (예: [제목](URL) 형식)
- '최신 해외학술 정보' 섹션: [“논문명”] 텍스트 앞에 하이퍼링크로 제공된 URL

## 추출 규칙

1. 'BOK 경제연구' 섹션에서 [[제XXXX-XX호] 패턴이 나타나면 새로운 아이템으로 추출
2. 제목은 [[제XXXX-XX호] 뒤의 영문 제목을 한국어로 번역하여 사용
3. 본문은 영문 요약을 자연스러운 한국어로 번역하여 300~500자로 요약
4. '최신 해외학술 정보' 섹션의 해외 논문은 제목만 제공되므로 본문 내용이 있는 경우에만 아이템으로 추출
5. 키워드는 논문 내용에서 추출하거나 원문 키워드를 참고하여 3~5개의 한국어 명사형으로 구성

- 'BOK 경제연구' 섹션은 한국은행 자체 연구 논문 소개, '최신 해외학술 정보' 섹션은 해외 주요 논문 소개로 이원화

- 동일한 구조가 매호 반복되므로, 이 분석 결과를 바탕으로 자동 추출 규칙 적용 가능
