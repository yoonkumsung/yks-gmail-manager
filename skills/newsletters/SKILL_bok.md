## 구조 분석

### 뉴스레터 특징
- 섹션 헤더(예: "최신 보고서", "경제상황 평가")로 구분
- 각 섹션 내 아이템은 제목과 설명 문단으로 구성
- 보도자료 섹션은 제목과 URL 쌍으로 나열

### 아이템 경계
- "## 섹션명" 형식의 헤더 다음 빈 줄 이후 첫 번째 문단
- [보도자료], [공지사항] 등 라벨로 시작하는 라인

### 제목 위치
- 섹션 헤더 직후 첫 번째 문단 시작 부분
- [보도자료] 등 라벨 뒤 제목 텍스트

### 본문 위치
- 제목 다음 줄부터 다음 섹션 시작 전까지의 문단
- ※ 참고 문구 이전까지의 내용

### 링크 위치
- 제목 텍스트 뒤 대괄호 안의 URL
- "자세한 내용은 <붙 임> 참조" 문구와 대응하는 보도자료 링크
