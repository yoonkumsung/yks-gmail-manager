## 구조 분석

### 뉴스레터 특징
- 단일 주제(스타트업 스케일업 전략)에 대한 여러 항목 나열
- 각 항목은 이모지(💡, ✅)로 시작하는 섹션 제목 아래에 번호/목록으로 구성
- 항목별로 제목(굵은 글씨)과 설명 문단, 일부는 URL 포함

### 아이템 경계
- 이모지(💡, ✅)로 시작하는 섹션 제목이 주요 구분자
- 각 섹션 내에서 번호(①, ②, ③) 또는 목표 번호(목표 1~5)로 개별 항목 구분
- 항목 제목은 '—' 또는 '→' 뒤에 나오는 텍스트

### 제목 위치
- 각 항목의 제목은 굵은 글씨(strong) 또는 이모지 뒤의 텍스트
- 예: '창업도약패키지', '벤처기업 확인제도', 'K-Startup (창업지원포털)'

### 본문 위치
- 제목 다음의 설명 문단 (여러 줄로 구성)
- 종료 신호: 다음 항목의 제목 또는 섹션 종료

### 링크 위치
- 각 항목 내 URL이 명시된 경우 (예: www.k-startup.go.kr)
- URL은 하이퍼링크 형태로 제공

---

## 추출 규칙

1. 각 섹션(💡, ✅) 내의 번호/목록 항목을 개별 아이템으로 추출
2. 제목은 굵은 글씨 또는 이모지 뒤의 핵심 명칭 사용
3. 요약은 설명 문단을 300~500자로 재구성
4. 링크가 명시된 경우만 포함, 없으면 빈 문자열
