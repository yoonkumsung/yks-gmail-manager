## 구조 분석

### 뉴스레터 특징
- Substack 플랫폼 기반 에세이형 뉴스레터
- 단일 주제 심층 에세이 (철학, 경영, 인문학 대가들의 사상 해석)
- 저자 개인의 통찰과 대가들의 인용문 결합
- 에세이 문체, 짧은 문단 구성

### 아이템 경계
- 하나의 에세이로 구성 (단일 아이템)
- 소제목이 있을 수 있으나 하나의 주제

### 제목 위치
- 이메일 제목이 에세이 제목
- 부제목이 본문 초반에 있을 수 있음

### 본문 위치
- 전체 본문이 하나의 에세이

### 링크 위치
- Substack 원문 링크 (본문 상단 또는 "View in browser" 링크)

## 추출 규칙

1. 전체를 1개 아이템으로 추출
2. 제목 20~50자
3. 요약 300~500자: 핵심 주장-인용된 대가/사상-저자의 해석-시사점
4. 키워드 3~5개 (인용된 인물명, 사상/개념명 포함)
5. 링크는 Substack 원문 URL
