## 구조 분석

### 뉴스레터 특징
- 미국 비즈니스/경제 뉴스 일간 다이제스트 (영문)
- **대문자 섹션 헤더**로 명확하게 구분
- 유머러스하고 캐주얼한 문체
- 본문이 길 수 있음 (15K자+)

### 섹션 구조 (대문자 헤더)
1. **MARKETS** — 시장 지표 (S&P, Nasdaq, 금리 등)
2. **[메인 기사 제목 - 대문자]** — 헤드라인 심층 분석 (예: TAKING A SUPER POSITION)
3. **READER POLL** — 독자 투표 (제외 대상)
4. **WORLD** — 국제 뉴스
5. **[두 번째 기사 - 대문자]** — 부가 심층 기사 (예: CLEAR THE WEIGH)
6. **[세 번째 기사]** — 추가 기사 (예: MALL UPDATE)
7. **QUIZ** — 퀴즈 (제외 대상)
8. **NEWS** — 짧은 뉴스 모음 (5~8개)
9. **RECS** — 추천/가벼운 콘텐츠

### 아이템 경계
- 대문자 섹션 헤더가 각 아이템/섹션의 시작
- NEWS 섹션 내 각 뉴스는 빈 줄로 구분

### 링크 위치
- 본문 내 하이퍼링크 (추적 URL 포함)

---

## 추출 규칙

1. **모든 대문자 섹션의 뉴스를 빠짐없이 추출** — MARKETS부터 NEWS까지
2. 메인 기사: 300~500자 한국어 요약 (핵심사실+수치+배경+시사점)
3. MARKETS: 주요 지표 변동 요약
4. NEWS 섹션: 각 뉴스 개별 아이템으로 추출 (짧은 요약 허용)
5. WORLD: 국제 뉴스 요약
6. 제목 20~50자 (한국어), 키워드 3~5개
7. **영문 → 한국어 번역 필수**
