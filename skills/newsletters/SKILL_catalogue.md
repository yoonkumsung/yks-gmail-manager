## 구조 분석

### 뉴스레터 특징
- 트렌드/라이프스타일 큐레이션 뉴스레터
- 해시태그(#) + 이모지로 카테고리 구분
- 각 카테고리에 1개 아이템 (제목 + 본문 설명)
- 짧고 캐주얼한 문체

### 카테고리 구조 (# + 이모지)
- #신상🍪 — 신제품 소개
- #패션🏃‍♀️ — 패션 트렌드
- #여름템☀️ — 계절 아이템
- #당충전🍪 — 식품/간식
- #맛집📘 — 맛집 추천
- #키링🍎 — 잡화/소품
- #패션⌚ — 액세서리
- #뉴스레터🎤 — 추천 뉴스레터
- #사이트☕️ — 추천 사이트/앱
- 기타 해시태그 카테고리 가능

### 아이템 경계
- 각 # 해시태그 줄이 새로운 아이템의 시작
- 해시태그 줄 → 본문(1~3문단) → 다음 해시태그

### 링크 위치
- 본문 내 대괄호 URL 또는 하이퍼링크 텍스트

---

## 추출 규칙

1. **모든 해시태그 카테고리의 아이템을 빠짐없이 추출**
2. 제목: 해시태그 뒤의 핵심 내용 (20~50자)
3. 요약: 제품명/브랜드명/특징을 포함. 짧은 항목은 짧은 요약 허용.
4. 키워드: 브랜드명, 카테고리, 제품 유형
5. 링크 추출

- 하단 구독/수신거부

## 특이사항

- 후반부 아이템(#뉴스레터, #사이트 등)이 누락되기 쉬움 — **마지막까지 모두 추출**
- 캐주얼한 문체를 요약에서 격식체로 변환
