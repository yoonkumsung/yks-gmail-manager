## 구조 분석

### 뉴스레터 특징
- 각 섹션은 간결한 제목과 한 줄 설명으로 구성됨
- 유아 활동과 보호자 정보를 함께 제공하는 형태
- 시각적 활동(색칠하기, 만들기)과 학습 활동(숫자, 글자)이 혼합

### 아이템 경계
- 각 아이템은 새로운 줄로 시작하는 짧은 제목(예: '정원 나들이 가요', '건강하게 먹어요')으로 구분됨
- 제목 바로 다음 줄에 해당 활동에 대한 한 줄 설명이 이어짐

### 제목 위치
- 텍스트 본문에서 새로운 줄로 시작하는 짧은 구문(예: '정원 나들이 가요')이 제목 역할을 함

### 본문 위치
- 제목 바로 다음 줄에 위치한 한 줄 설명 텍스트 (예: '정원에 핀 예쁜 꽃을 색칠하고 땅속 개미 친구도 만나 보아요')

### 링크 위치
- 이 뉴스레터 본문에는 명시적인 URL 하이퍼링크가 포함되어 있지 않음. 정보는 텍스트 내에 모두 제공됨.

## 추출 규칙

1. 새로운 줄로 시작하는 짧은 구문을 아이템 제목으로 식별
2. 제목 다음 줄의 텍스트를 본문/요약으로 추출
3. 본문 텍스트를 바탕으로 키워드를 추출
4. 링크는 원문에 없으므로 빈 문자열로 처리

## 제외 영역

- 뉴스레터 상단의 인사말 및 소개 문단 ('서울시 유아 신문...담겨 있어요.')
- 뉴스레터 하단의 발행처 정보 ('서울특별시...문의 120')

## 특이사항

원문에 외부 링크가 없으며, 모든 콘텐츠가 텍스트 내에 포함되어 있음. 각 아이템은 유아 활동 또는 관련 정보를 매우 간략하게 소개하는 형태.
