## 구조 분석

### 뉴스레터 특징
- 테이블(표) 형식으로 연구 논문/보고서 목록을 나열
- 섹션 헤더(논단, 금융포커스, 해외 경제 등)로 카테고리 구분
- 각 행에 논문/보고서 제목이 포함

### 아이템 경계
- 테이블의 각 행이 하나의 아이템 (섹션 헤더 행 제외)
- 섹션 헤더: "논단", "금융포커스", "해외 경제 및 금융 이슈", "주요 경제·금융동향" 등은 카테고리 라벨이므로 아이템이 아님
- 논문/보고서 제목이 포함된 행만 아이템으로 추출

### 제목 위치
- 테이블 행의 텍스트 자체가 제목

### 본문 위치
- 본문 없음 (제목만 있는 목록형). 제목이 곧 내용.

### 링크 위치
- 각 행에 포함된 URL (있는 경우)

1. 테이블 행에서 논문/보고서 제목을 아이템으로 추출
2. 섹션 헤더 행("논단", "금융포커스", "해외 경제 및 금융 이슈", "주요 경제·금융동향")은 아이템이 아닌 카테고리 라벨이므로 제외
