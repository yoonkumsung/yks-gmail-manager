## 구조 분석

### 뉴스레터 특징
- 유네스코 한국위원회 공식 뉴스레터
- 단일 주제 심층 기사 형태 (교육, 문화, 과학 등 유네스코 관련 주제)
- 학술적이면서도 대중적인 문체
- 스티비(Stibee) 플랫폼 기반 발송

### 아이템 경계
- 하나의 메인 기사로 구성
- 기사 내 소제목으로 섹션 구분

### 제목 위치
- 이메일 제목이 기사 제목
- 본문 내 부제목이 있을 수 있음

### 본문 위치
- 전체 본문이 하나의 기사

### 링크 위치
- "뉴스룸에서 보기" 링크로 원문 제공 (stibee 리다이렉트 URL)

## 추출 규칙

1. 전체를 1개 아이템으로 추출
2. 제목 20~50자, 핵심 주제 + 키워드
3. 요약 300~500자: 주제-핵심 데이터-유네스코 입장-시사점 구조
4. 키워드 3~5개 (유네스코, SDG 관련 용어 포함)
5. 링크는 "뉴스룸에서 보기" URL 추출
