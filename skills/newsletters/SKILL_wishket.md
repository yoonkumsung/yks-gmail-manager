## 구조 분석

### 뉴스레터 특징
- 제목과 링크가 함께 표시
- 각 아이템에 카테고리와 소요시간 표시
- 하단에 회사 소개 및 구독 해지 링크 포함
- 아이템 사이에 빈 줄 구분

### 아이템 경계
- 각 아이템은 카테고리(예: 개발, 프로덕트)와 소요시간(예: 9분, 5분) 라인 뒤에 본문 텍스트가 이어지는 형태이며, 아이템 사이에 빈 줄과 구분선(---)이 없음

### 제목 위치
- 제목은 아이템 시작 부분에 위치하며, 링크와 함께 대괄호 안에 표시됨

### 본문 위치
- 본문은 제목과 링크 뒤에 이어지는 p 태그 형태의 텍스트이며, 카테고리 및 소요시간 라인 뒤에 위치

### 링크 위치
- 링크는 제목 뒤에 대괄호 안에 삽입된 URL이며, 클릭 가능한 하이퍼링크 형태

## 추출 규칙

1. 제목은 아이템 시작 라인에 있는 텍스트와 링크를 분리
2. 요약은 본문 텍스트에서 핵심 내용 추출
3. 키워드는 명사형 추출
4. 링크는 대괄호 안의 URL 사용
5. 제외 영역은 헤더, 푸터, 광고 등
