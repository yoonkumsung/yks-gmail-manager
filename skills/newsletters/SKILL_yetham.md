## 구조 분석

### 뉴스레터 특징
- 에너지 바(🔋)와 하트로 진행 상황 시각화
- 각 아이템 시작에 고유 이모지 식별자 사용(👀, 💭, ☘️)
- 익명 구독자 고민과 멘토 답변 구조화

### 아이템 경계
- 이모지(👀, 💭, ☘️)로 시작하는 섹션 제목 라인
- '🔋 마음 에너지' 상태 표시줄로 아이템 구분

### 제목 위치
- 이모지 바로 뒤 텍스트 (예: '👀 슬프고 힘들 때, 파리가 되어보라고?! _ 벽에 붙은 파리 효과')
- 밑줄(_) 또는 이모지로 강조된 제목 구간

### 본문 위치
- 제목 다음 줄부터 다음 '🔋' 상태 표시줄 직전까지
- '➡️' 또는 '🧚‍♀️' 같은 CTA 심볼 이전 본문 내용

### 링크 위치
- '예담이네 블로그 방문하기' 텍스트 뒤 URL
- '힐링 커뮤니티 참여하기' 라벨과 연동된 하이퍼링크

## 추출 규칙

1. 이모지 식별자로 시작하는 섹션을 개별 아이템으로 처리
2. 익명 고민과 멘토 답변을 하나의 아이템으로 통합
3. 에너지 바(🔋) 상태 변화 지점에서 아이템 경계 설정
