## 구조 분석

### 뉴스레터 특징
- 개인 경험 기반의 휠체어 안내서
- 교육적 목적의 심층 분석 구조
- 서사적 스토리텔링과 정보 전달 결합

### 아이템 경계
- 단일 주제에 대한 연속적인 설명
- 소제목으로 '휠체어의 기초', '휠체어 실전' 구분

### 제목 위치
- 메인 제목: '휠체어의 기초부터 심화까지'
- 서브 제목: '휠체어의 기초 - 구분하기' 등 섹션별 소제목

### 본문 위치
- 서술형 본문 전체 (개인 경험 → 일반 정보 전개)
- 목록 형식의 설명 ('수동휠체어:', '전동휠체어:' 등)

### 링크 위치
- 본문 내 직접 링크 없음
- 하단에 '봄 시즌 활동 알림 받기' 외부 링크만 존재

## 추출 규칙

1. 단일 아이템 처리
2. 제목 = 메일 제목 + 본문 주제 결합
3. 요약 = 핵심 정보 3요소(종류/특징/구매) 포함
