## 구조 분석

### 뉴스레터 특징
- [IMAGE:] 로고와 섹션 제목(공연/전시/미술/교육/체험)으로 분류된 3개 섹션
- 테이블 형식으로 여러 행사를 정렬
- 각 행사는 제목(하이퍼링크), 날짜, 장소, 가격, 연락처 정보로 구성
- 월간 문화달력 포맷(7월)

### 아이템 경계
- 각 행사는 테이블의 개별 셀 단위
- 빈 행(| | |)이 행사 간 구분
- 섹션 경계: [IMAGE: 공연], [IMAGE: 전시/미술], [IMAGE: 교육/체험] 이미지 텍스트로 명확히 구분

### 제목 위치
- 대괄호로 감싼 텍스트 + 기관명이 행사 제목
- 예: [세종문화회관 천하제일탈공작소, baan on Sync Next 26]
- 제목 전체가 하이퍼링크

### 본문 위치
- 제목 다음 줄: 날짜(YYYY-MM-DD(요일) ~ YYYY-MM-DD(요일), 시간)
- 그 다음 줄: 장소(기관명 + 상세 장소명)
- 가격 정보('무료' 또는 '석 가격원' 형식)
- 마지막: 연락처(전화번호)
- 본문 설명 없음

### 링크 위치
- 제목의 하이퍼링크
- npcrm.eseoul.go.kr로 시작하는 추적 URL
- 각 행사별로 고유 URL

## 추출 규칙

1. 섹션별 그룹화(공연/전시/교육)
2. 각 테이블 셀이 하나의 행사 항목
3. 제목: 대괄호 내 텍스트
4. 날짜: YYYY-MM-DD(요일) ~ YYYY-MM-DD(요일) 형식, 시간 포함
5. 장소: 기관명 + 세부 장소명
6. 가격: '무료' 또는 계층별 가격
7. 연락처: 전화번호 추출
