## 6. 심각 문제 (우선 수정)

1. **[중대 번역오류] FL-3 "영국 군부 방문"** — `British monarch`(영국 군주)를 "군부"로 오역, 의미 왜곡. 추출/번역 프롬프트에 monarch·군주 처리 보강 필요.
2. **[중복 미병합] AM↔PM 유가/휘발유 동일 사건이 3개 아이템으로 분산** — `duplicates_removed: 0`. cross-newsletter 의미기반 dedup 필요(link 도메인만으론 미탐지). AM/PM 시점차 수치는 병합 시 양쪽 보존 권장.
3. **[정크] "큐" 토큰** — merged AM판본 찰리 커크 요약에 정체불명 토큰 삽입. 파싱/정제 단계 점검.
4. **[출처 오매핑] AM 왕좌게임·$700억 동일 URL** — 서로 다른 기사인데 link 중복. 본문 링크-아이템 매핑 로직 점검.

> **사용자 "텍사스 중복" 지적은 미국_시사 라벨에는 해당 없음**(텍사스 1회 정상 등장). 다른 라벨(미국_경제 등) 또는 라벨 간 교차 중복을 별도 확인 권장.
