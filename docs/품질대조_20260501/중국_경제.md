## 4. 구체 문제 목록

### (a) 정크 한줄/홍보성 — 전체 15건 (대표)
- "해외 바이어 증명서 간편 등록 서비스 개시" — 등록 절차 안내
- "'공정2도시' 지하철 0.01위안 승차권 혜택" — 프로모션
- "'i-Invite' 보상 및 기회 제공 프로그램" — 프로모션
- "바이어 눈으로 본 광교회: 광교회의 신자/눈/충성 고객" — 인터뷰 티저 3건

### (b) 번역 문제 (경미)
1. **"광교회"** — Canton Fair의 중국어 명칭은 "광교회(廣交會)"가 맞으나, 한국 독자에게는
   "광저우 수출입상품교역회" 또는 "캔톤페어"가 더 자연스럽다. 직역체로 일관.
2. **"'이민증 해석' 온라인 도착 카드 가이드"** — 원문 "答移解问"(이민 Q&A)의 오역.
   "이민증 해석"은 의미 불명. "출입국 Q&A / 입국카드 안내"가 정확.
3. **"공정2도시"** — "Fair2City"의 직역 오역. 고유명사이므로 음역 유지가 옳음.

### (c) 출처 — 정상

(`/x/c/?...`) 그대로 보존 — 클릭 추적 URL이지만 원문 링크로 유효.
