# 리포트 UI 스펙 (REPORT_UI)

뉴스레터 다이제스트 HTML 리포트의 단일 디자인·동작 기준. MD/HTML 출력이 이 문서를 공통 기준으로 따른다.
(과거 "목록형 토글"이 MD에만 구현되고 HTML엔 누락된 불일치를 막기 위함)

## 설계 원칙

1. **가독성 최우선** — 본문 16px+, line-height 1.6(CJK), 넉넉한 여백. 한 화면에 너무 많은 카드를 쌓지 않는다.
2. **토스(Toss) 스타일** — 흰 배경, 부드러운 라운드(16~20px), 얕은 그림자/1px 경계, 파란 포인트, 명확한 위계, 군더더기 없음.
3. **정보 누락 제로** — 어떤 아이템도 화면에서 사라지지 않는다(접기는 OK, 삭제는 X).
4. **모든 아이템에 이동 수단** — `원문 보기`(link 있을 때) 또는 `Gmail에서 보기`(message_id 기반) 버튼이 **항상 최소 1개** 존재.
5. **접근성** — WCAG 2.2 AA, 터치 타겟 24px+, focus 가시, `prefers-reduced-motion` 존중.

## 디자인 토큰

- **폰트**: Pretendard Variable (CDN), 폴백 system-ui.
- **색**:
  - 배경 `#FFFFFF`, 보조배경 `#F2F4F6`, 경계 `#E5E8EB`
  - 텍스트 `#191F28`(본문), `#4E5968`(보조), `#8B95A1`(메타)
  - 포인트(토스블루) `#3182F6`, 포인트연 `#E8F3FF`
  - 라벨색: 기존 `LABEL_COLORS`(generate_html.js) 재사용 — 라벨 점/탭에만 사용
- **라운드**: 카드 18px, 칩/버튼 10px, pill 999px
- **그림자**: `0 1px 3px rgba(0,0,0,.04), 0 1px 2px rgba(0,0,0,.06)`
- **간격 스케일**: 4/8/12/16/24/32
- **타입스케일**: 헤더 20px·700 / 카드 제목 17px·600 / 본문 15.5px·400·lh1.65 / 메타 13px / 칩 12.5px

## 레이아웃

- **헤더(sticky)**: 좌측 "오늘의 뉴스레터" + 날짜, 우측 총 건수 배지 + 검색 입력. 얇은 하단 경계.
- **라벨 내비(sticky, 헤더 아래)**: pill 탭(라벨색 점 + 이름 + 건수). 가로 스크롤(모바일). 클릭 시 해당 라벨 섹션만 표시.
- **본문**: 선택된 라벨 섹션. 라벨 상단에 작은 통계줄(N개 · 중복 M 제거).

## 아이템 표시 규칙

### 일반 라벨 (기본)
각 아이템 = **카드**:
- 제목(17px·600) — 클릭 시 link 있으면 원문 새 탭.
- 요약(15.5px, lh1.65). 400자↑이면 "더 보기"로 접기(4줄 클램프).
- 키워드 칩(보이게, `#키워드`, 최대 6개) — **표시 ON**.
- 하단 메타줄: 출처(source) · (있으면)수신시각, 우측 버튼 `원문 보기`(파란 채움) / `Gmail에서 보기`(회색 외곽). 화살표(↗) **금지**.

### 목록형 라벨 (자동 감지)
판정: `items.length >= 30 && 평균 summary 길이 < 100` (MD의 `isListType`와 동일 기준).
- 상단 **"주요 동향" 인사이트 카드**: `clusterItemsByKeyword`(threshold 0.15)로 묶어 대표 제목 "외 N건"을 5~8개 노출.
- **"전체 N건 펼치기" 토글**(`<details>` 또는 버튼+JS): 펼치면 **한 줄 리스트**.
  - 한 줄 = 제목(굵게) + (있으면 첫 문장 ~60자 회색) + 우측 미니 버튼(원문/Gmail).
  - 줄 사이 1px 구분선, 호버 시 옅은 배경.
- 목록형도 누락 제로: 모든 아이템이 펼친 리스트에 포함.

## 동작 (client JS)

- **검색**: 제목·요약·키워드·출처 대상 즉시 필터. Cmd/Ctrl+K 포커스. 매칭 수 표시.
- **라벨 전환**: pill 클릭 → 해당 섹션만 노출(나머지 hidden).
- **목록형 토글**: 펼침/접힘. 기본 접힘.
- **더 보기**(긴 요약): 클램프 해제.
- 모션 150~200ms ease, reduced-motion 시 제거.

## 비고

- 키워드는 화면에 **표시**한다(검색 전용 숨김 정책 폐기).
- 이 문서와 구현(`scripts/render_report.js`)이 어긋나면 문서를 우선해 구현을 고친다.
