## 6. 중복 — 과분할 + 병합후 잔존 (이 라벨 최대 이슈, 중간)

### 6-A. 추출 단계 과분할(정상적 설계 부산물)
같은 제품을 "뉴스레터 티저 + 크롤링 본문 상세"로 2회 추출하는 패턴이 다수. 예:
- 휴고 스프리츠: 975f #13(티저 36자) + #42(상세 302자)
- 메리제인: 966e #2 티저 + #37(토리버치 상세) + #42(플랫 가이드)
- 립스틱: 966e #8 티저 + #37 가이드 + #38 6종 + #39/#40 개별 랑콤 2종
- 비데: 966e #15 티저 + #56 S5 + #57 C5 + #58 S2 + #59/#60 브론델 — 1개 가이드가 6아이템으로 분할
- 손전등: 966e #4 티저 + #30 가이드 + #31/#32/#33 개별 3모델

→ 일부는 **정당한 분리**(개별 제품·가격이 다른 단신)이나, 티저+상세는 **동일사건**이므로 병합 대상.

### 6-B. 병합후 merged에 동일사건 잔존(누락 아닌 중복 실패)
