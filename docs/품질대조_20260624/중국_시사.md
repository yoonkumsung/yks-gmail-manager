| 형태 | 메일 수 | 추출 n | 구조 | 요약 성격 |
|---|---|---|---|---|
| **A. 링크목록 브리핑** (Daily Briefing / Markets / Opinion / Style / PostMag) | 14 | 10~58 | 섹션헤더 + 헤드라인-링크. **본문 없음.** 일부는 `=== 원문 기사 전문 ===`로 featured 기사 풀텍스트 크롤 부착 | 헤드라인-only 패러프레이즈가 다수(짧은 게 정상) |

## 근본원인 분류 (룰 vs LLM)

**LLM이 풀 문제**
1. 🔴 **할루 #42** (중국 주체·칼슨 인물 날조) — `link:""` 헤드라인-only에서 배경 창작. 프롬프트에 "헤드라인/본문에 없는 주체·인물·발언 생성 절대 금지" 강화.
2. 🔴 **번역**: 제목 영문 금지, Starmer "goes"류 의미검수, 음역 일관성 — **SCMP SKILL에 번역규칙 신설**(현재 결핍 확인됨).
3. 🟡 동일 칼럼 문단 과분할 → 병합 LLM이 의미 병합.

**룰베이스(코드·JSON)가 풀 문제**
4. 🔴 **LINK 무결성** (최우선): 풀텍스트 아이템의 article-id/URL 매핑 버그(off-by-one·inline URL 채택·공란). markdown 링크 추출 로직 점검. 이게 5번 under-merge의 직접 원인.
5. 🔴 **중복**: (a)B형 `tc=2`/`tc=3` 동일 URL 이중 크롤 → article-id dedup, (b)헤드라인판/풀텍스트판/토픽 census판 3중 추출 정리, (c)공란 링크는 제목 유사도 등 보조 키로 병합.
6. 🟡 **정크 제거**: 토픽 컬렉션 헤더("2026 Trump-Xi summit" 슬러그) + 헤드라인 truncated fragment(상위 아이템과 동일 LINK) 룰 제거.
7. 🟡 **토픽 census 오염**: 크롤 풀텍스트 앞단의 SCMP 토픽 위젯/홈페이지 관련기사(날짜 5/14~6/16 혼입)를 "오늘 뉴스"로 추출 — 크롤 범위 경계 룰 필요.
8. 🟡 **금지표현 목록**에 메타 회피("본문이 짧아…", "본문 일부만 공개…") 추가.

> **결론**: 이 라벨은 **누락 제로·오병합 제로를 달성**했고 SCMP 콘텐츠 커버리지는 견고하다. 실패는 (1) **링크 무결성 붕괴 → 중복 under-merge 90건**(룰), (2) **번역규칙 결핍**(LLM/SKILL), (3) **할루 1건**(LLM)에 집중된다. 우선순위는 **링크 매핑 룰 수정 → 중복 dedup 보강 → SCMP 번역 SKILL 신설 → 할루 가드 강화** 순.
