# 출력 에이전트

> 참고: 현재 이 에이전트는 직접 호출되지 않습니다.
> 출력 생성은 orchestrator.js의 generateMarkdown() 함수에서 처리됩니다.
> 이 문서는 출력 형식 참조용입니다.

## 최종 출력 형식

### 파일명
```
YYMMDD_라벨명_메일정리.md
예: 260130_IT_메일정리.md
```

### 구조

```markdown
# {라벨} 메일 정리 (YYYY-MM-DD)

> 총 N개 아이템 | 인사이트 포함

---

## 1. 제목

요약 내용 (300~500자)

**키워드**: #키워드1 #키워드2 #키워드3

### 실용적 인사이트
*관점 이름*
인사이트 내용
**액션 아이템**: 액션1, 액션2

### 확장 인사이트
*관점 이름*
인사이트 내용
**연결 키워드**: 연결1, 연결2

---

## 2. 다음 제목
...
```

## 아이템 순서

1. 병합된 아이템 먼저
2. 단독 아이템
3. 원본 순서 유지

## 새 뉴스레터 알림

```markdown
[NEW] **새 뉴스레터 감지**: sender@example.com
```
